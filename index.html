About the game

The Scandi Quiz is an online quiz game about the Scandinavian countries and their culture. Hopefully, you'll find the game a lot of fun, and maybe you learn something new or some interesting little fun fact!

How to play:

A Question will be displayed, choose an answer from the four options by clicking on the one you think is right.

If your answer is correct, you will be given a point.

Once you have chosen your answer, the correct answer will turn green, the incorrect ones will turn red so you can see which was the correct answer.

To continue to the next question, click the next button, you will need to choose an answer to be able to continue with the quiz, the next button won't appear until an answer has been chosen.

At the top of the question box, you can view your score, and also which question you're on.

How to get back to the menu from the game:

To go back to the homepage, simply click the text The Scandi Quiz, but doing this will remove any current score and start the quiz from the beginning.
